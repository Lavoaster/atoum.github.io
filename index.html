atoum


atoum is a simple, modern and intuitive unit testing framework for PHP! It is completly standalone, everything is available and works out of the box:

  • Tests isolation and parallelization to get better performances and avoid side-effects on tests,
  • Full-featured mock engine,
  • Adapters to fake PHP functions,
  • Virtual streams to mock filesystem,
  • And many more!

Use it

Installing atoum in your project is as simple as that:

{% highlight sh %} composer require --dev 'atoum/atoum' vendor/bin/atoum -d tests/units {% endhighlight %}
